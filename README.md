# Ethical-Hacking-Commands-Generator
A static, single-page web utility for generating complex command-line syntax for ethical hacking and security auditing tools.

## üìã Overview

**CmdGen** is a lightweight, responsive web application designed to help penetration testers, CTF players, and security researchers quickly construct accurate commands for popular security tools. Instead of memorizing obscure flags or constantly checking man pages, users can select a tool, configure parameters via a GUI, and instantly get the production-ready command.

It runs entirely in the browser using vanilla JavaScript and Tailwind CSS. No backend or compilation is required.

## ‚ú® Features

- **20+ Integrated Tools:** Covers the entire kill chain from Reconnaissance to Exploitation.
    
- **Dynamic Configuration:** Forms adapt based on selected modes (e.g., changing Nmap scan types or Metasploit payloads).
    
- **Real-time Generation:** The terminal preview updates instantly as you type.
    
- **One-Click Copy:** Quickly grab the generated command for your terminal.
    
- **Hacker Aesthetic:** Fully responsive dark mode interface with neon accents.
    
- **Zero Dependencies:** Single HTML file architecture. Just download and run.

## üöÄ Installation & Usage

Since this is a static website, "installation" is as simple as opening a file.

1.  **Clone the repository:**
    
    ```
    git clone https://github.com/yottajunaid/Ethical-Hacking-Commands-Generator.git
    ```
    
2.  **Navigate to the directory:**
    
    ```
    cd Ethical-Hacking-Commands-Generator
    ```
    
3.  **Run:**
    
    - Simply double-click `index.html` to open it in your browser.
        
    - **OR** serve it locally with python:
        
        ```
        python3 -m http.server 8000
        ```
